Welcome!

Baatvakta (Norwegian for 'Boat Watch'): A project to check various parameters on a boat, such as temperatures, battery voltages, pump activities etc. Actual data is sendt via GPRS to a server and logged. In case of alarming low/high values an SMS is sendt to inform the owner of the situation. The project started as a student's project at NTNU (Norwegian University of Technology and Natural Science).

This site shows data for the boat Veslefrikk
